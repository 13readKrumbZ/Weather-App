This app was created using nodejs and react to get a better understanding and a practical exercise on how APIs work. 
It's a simple web page that gives the weather information for locations in Guyana.
How to run. 
This code is use Node.js and react 
Please download Node from the website :
 https://www.google.com/url?sa=i&source=web&rct=j&url=https://nodejs.org/en&ved=2ahUKEwipr6Pu8aqSAxXrRTABHVDhLhYQy_kOegYIAQgFEAE&opi=89978449&cd&psig=AOvVaw0LXgjfspgAhIw8ki0s5gy2&ust=1769574771008000

Since this program uses nodejs, if you don't have it installed, please get it from the official website https://nodejs.org/ 
Then open the weatherapp folder in your code editor and change directory(cd weather_app) to weather_app and install nodejs (npm install)
Since the program uses a free API that uses unique keys for each user, please create an account at 
https://home.openweathermap.org/api_keys. 

replace the default "API_Key" variable in the Weather App\weather_app\src\app\page.jsvfile

Paste command in terminal 
1 ( cd weather_map ) 
2 ( npm run dev )

app runs on http://localhost:3000
enter Linden or Georgetown into search bar.
